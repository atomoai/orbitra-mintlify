# `orbitra.lake.services.saved_query`

## Classes

### `SavedQueryService`


Service class for managing saved query operations.

**Attributes:**
- `raw_service`: The raw service for saved query operations.
- `db_service`: The database service for saved query operations.


**Methods:**

```python
create_or_update_saved_query(self, namespace: str, saved_query: SavedQueryCreate) -> SavedQuerySchema
```

Create a new saved query or update an existing one, saving the old version to history.


#### `delete_saved_query`

```python
delete_saved_query(self, namespace: str, saved_query_name: str) -> None
```

Delete a saved query's current definition, preserving its version history.


#### `get_saved_query`

```python
get_saved_query(self, namespace: str, saved_query_name: str, version: Optional[datetime] = None) -> SavedQuerySchema
```

Retrieve a saved query by name, optionally at a specific historical version.


#### `get_saved_query_data`

```python
get_saved_query_data(self, namespace: str, saved_query_name: str, scan_filters: List[Filter], limit: Optional[int] = None, engine: Literal['local', 'remote'] = 'local') -> pd.DataFrame
```

Execute a saved query's SQL and return the results.

Wraps the saved query SQL in a subquery and applies filters/limit on top.

**Args:**
- `namespace`: The namespace containing the saved query.
- `saved_query_name`: The name of the saved query to execute.
- `scan_filters`: Filters to apply on top of the result.
- `limit`: Maximum number of rows to return.
- `engine`: Query engine to use. Defaults to "local".

**Returns:**
- pd.DataFrame: The query result as a DataFrame.

**Raises:**
- `LakeError`: If the saved query does not exist or if query execution fails.
