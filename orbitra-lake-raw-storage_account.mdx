# `orbitra.lake.raw.storage_account`

## Classes

### `StorageAccountRawProvider`


Azure Blob Storage implementation of BaseRawProvider.

This provider persists pandas DataFrames as Parquet files directly
into an Azure Blob Storage container. It requires a valid Azure
`TokenCredential` for authentication and Orbitra Lake settings to
resolve the target account. The container name is provided per
operation (read/write).

**Attributes:**
- `settings`: Orbitra Lake configuration containing
the storage account URL.
- `Credential`: Azure credential used to authenticate
against the Blob Storage account.
