# `orbitra.lake.raw.local`

## Classes

### `LocalRawProvider`


Local filesystem implementation of BaseRawProvider.

This class provides a local alternative to cloud-based raw data
storage. Instead of uploading data to a remote Blob Storage,
DataFrames are persisted as Parquet files on the local filesystem.

**Attributes:**
- `settings`: Orbitra Lake settings object that must
include the attribute `orbitra_lake_local_path` to define the
base directory for raw storage.
