## Classes

### `LocalDB`


Concrete implementation of PyIcebergDB for local SQLite catalog.


**Methods:**

#### `close`

```python
close(self) -> None
```

Close the SQLite catalog connection to release file locks.

#### `refresh_token`

```python
refresh_token(self, token: str) -> None
```
